# Set up

1. Install [node.js](http://nodejs.org/download/)
2. Install [foreman](http://ddollar.github.io/foreman/) ```gem install foreman```
3. Run the app ```foreman start```


# Environment variables

* ```PORT```
* ```REDIS_PORT```
* ```REDIS_HOST```
* ```ERROR_URL```
* ```PARAM```
* ```REFERER_PARAM```
